# Strato Studio — Landing Page (Build)

Landing de la agencia con las **5 etapas** como núcleo: Estratégica → Producción estructurada → Postproducción → Distribución inteligente → Optimización continua.

## Cómo ejecutar

```bash
npm install
npm run dev
```

Abre [http://localhost:5173](http://localhost:5173).

## Build para producción

```bash
npm run build
npm run preview
```

## Estructura

- **Carpeta de modelado** (Figma/prototipo): `Strato Studio Landing Page (1)`
- **Carpeta de build** (este proyecto): `strato-landing-build`

Stack: Vite + React 18 + TypeScript + Tailwind CSS.
